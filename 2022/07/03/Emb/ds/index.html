栈的实现原理

数组 + 循环 -》 栈 队列

栈

FILO (First Input Last Output)

  1. 栈内容成员可以是任意数据类型
  2. 栈操作：
    1. 入栈 出栈 判断栈是否为空
    2. 栈顶指针指向顶部空白内存

栈的应用

  1. 中缀表示法 (1 + 2) * (3 - 4)
  2. 逆波兰表示法 12+34-*

中缀转换后缀表示法

  1. 从左至右来处理数据
  2. 左括号忽略
  3. 数值直接输出，操作法则入栈
  4. 右括号出栈操作法

十进制转二进制

  1. 判断n是否为零，非零则%2，将值入栈，再更新n = / 2值
  2. 重复操作，直到n为零
  3. 只要栈顶不为空，就出栈

回文判定

  1. abba ab ba
  2. abdba ab d ba
  3. 取出字符串长度， 并将前半部入栈
  4. 如果栈不为空，取出栈顶元素与后半比较
  5. 长度若为奇数，则跳过中间元素比较
